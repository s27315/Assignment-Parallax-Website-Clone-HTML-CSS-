# Parallax Website Clone

## Description
A single-page parallax scrolling website built as a clone of the W3Schools Parallax Template.
The theme is **"Explore Earth"** — a travel destination showcase featuring stunning landscapes,
a travel gallery, destination facts, and a contact section.

## Features
- Fixed/sticky navigation bar with mobile hamburger menu
- 3 parallax sections using `background-attachment: fixed`
- Semantic HTML5 structure (`<nav>`, `<section>`, `<footer>`, `<form>`)
- Travel gallery grid with hover effects
- HTML Table (Travel Facts with destinations, continents, seasons, temperatures)
- Unordered List (Top Destinations)
- Contact form with name, email, and message fields
- Google Maps iframe embedded in the Contact section
- Font Awesome icons throughout
- Google Fonts (Montserrat + Lora)
- Fully responsive layout with CSS media queries
- External stylesheet only (no inline styles)

## HTML Elements Used
- Semantic tags: `<nav>`, `<section>`, `<footer>`, `<header>`
- `<table>`, `<thead>`, `<tbody>`, `<tr>`, `<th>`, `<td>`
- `<ul>`, `<li>` (destination list)
- `<form>`, `<input>`, `<textarea>`, `<button>`, `<label>`
- `<iframe>` (Google Map)
- `<img>` (gallery cards)

## CSS Highlights
- `background-attachment: fixed` for parallax effect
- 5+ CSS classes: `.parallax`, `.gallery-card`, `.contact-form`, `.contact-info`, `.facts-table`, `.btn-primary`
- Pseudo-classes: `:hover` (nav links, buttons, gallery cards, table rows, social links), `:focus` (form inputs), `:nth-child` (table rows)
- Media queries for tablet (≤900px) and mobile (≤600px)
